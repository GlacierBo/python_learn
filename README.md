## Python

接触 python 最直观的感受就是 python 真的是无所不能。

它涉及的面很广：网络编程、数据库、多媒体编程、科学计算、windows编程、企业与政务应用，同样包括目前很火的云计算领域等。

python 语法简洁，简单，明确，丰富的第三方库。越来越多的人投入 python 的怀抱，为 python 贡献自己的一份力量，十几年的沉淀让 python 变得越来越优秀。

纸上得来终觉浅，绝知此事要躬行。
